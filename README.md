# Nicholas Chicirda's Technical Assessment for MOD

## To run locally (Linux)  
### Ensure you have node and npm installed  
```
    sudo apt install nodejs  
    sudo apt install npm  
```
### Verify Node and npm  
```
    npm -v  
    node -v  
```
### Run the app  
```
    npm run start  
```
### Go to localhost:3000 on browser
